I build multi-agent AI systems and cloud-native backends that run in production.

Most of my work sits at the intersection of autonomous agent orchestration (LangGraph, CrewAI), retrieval-augmented generation, and infrastructure (FastAPI, Docker, AWS, Azure). I've shipped agent pipelines with up to 18 coordinated agents serving real users, and I deploy them behind Traefik with subdomain-based routing on managed cloud infrastructure.

I also teach what I build. I run [The College of AI](https://www.youtube.com/@TheCollegeOfAI) on YouTube and have taught Python, Agentic AI, Docker, and N8N to 500+ students across multiple cohorts.

### Currently working on

- Multi-agent orchestration systems using LangGraph with complex state management and human-in-the-loop workflows
- Cloud-native service deployments with FastAPI, Docker, Traefik, and AWS (EC2, RDS, Route53, ACM)
- RAG pipelines with custom chunking strategies across FAISS, Qdrant, and ChromaDB
- Workflow automation with N8N for content repurposing and business process orchestration
